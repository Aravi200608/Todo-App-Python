# To-Do App (Python CLI)

## Description
A simple command-line To-Do application built using Python.
Features:
- Add tasks
- View tasks
- Mark tasks as done/undone
- Delete tasks
- Persistent storage using JSON

## How to Run
```bash
python main.py
```

## Files
- main.py : Application logic
- tasks.json : Auto-created to store tasks

## Concepts Used
- Lists & Dictionaries
- File handling
- Functions
